# Complete C Programming

A structured C programming learning repository covering fundamental to advanced concepts with unit testing.

## Overview

This project provides a comprehensive path through C programming, organized by topics from basics to advanced concepts. Each topic includes practical code examples and unit tests using the Unity testing framework.

## Features

- Progressive learning structure from basic to advanced topics
- Hands-on code examples for each concept
- Unit tests to verify understanding and code correctness
- Unity test framework integration

## Topics Covered

- Data types and variables
- Source preprocessing
- Control structures
- Functions and pointers
- Memory management
- And more...

## Getting Started

1. Clone the repository
2. Navigate to the topic folder you want to study
3. Read the source files and run the examples
4. Run the associated unit tests to verify your understanding
